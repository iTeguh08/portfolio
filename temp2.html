Educational game Fruit Quiz

Dalam proyek final ini, saya mengembangkan sebuah game kuis buah yang dirancang khusus untuk mengedukasi anak-anak tentang berbagai jenis buah dengan cara yang menyenangkan. Menggunakan ActionScript 3.0 dengan mekanisme dalam game yaitu Drag and Drop, dan Button, proyek ini merupakan hasil dari perjalanan panjang melalui puluhan proyek sebelumnya, yang memberikan saya fondasi yang kuat dalam pemrograman dan desain game. Game ini terdiri dari beberapa scene interaktif yang memudahkan anak-anak belajar sambil bermain. Di awal permainan, terdapat scene loading yang menyajikan animasi yang menarik dan musik latar yang menyenangkan, membantu membangun antisipasi sebelum memasuki menu utama. Menu utama menawarkan dua pilihan permainan: "Berhitung" dan "Tebak Gambar," masing-masing dilengkapi dengan instruksi yang jelas. Pada mode "Berhitung," anak-anak dapat menjawab soal matematika sederhana, sementara di mode "Tebak Gambar," mereka harus mencocokkan gambar buah dengan nama yang benar. Setiap interaksi dilengkapi dengan efek suara yang memberikan umpan balik positif atau negatif, membantu anak-anak memahami jawaban mereka. Notifikasi visual, seperti tanda centang atau silang, menambah elemen edukasi dengan cara yang menyenangkan. Selain itu, timer digunakan untuk memberikan tantangan dan meningkatkan fokus anak-anak selama permainan. Dengan fitur-fitur ini, game kuis buah ini tidak hanya menghibur tetapi juga menjadi alat pembelajaran yang efektif, memberikan pengalaman belajar yang menyenangkan dan menarik bagi anak-anak. Proyek ini merupakan langkah besar dalam pengembangan kemampuan saya dalam menggunakan ActionScript 3.0 dan menciptakan aplikasi yang interaktif serta mendidik.

GUI Movie Website with API

Dalam project ini, saya mengembangkan tampilan web bioskop yang menarik dan interaktif dengan menggunakan data yang diambil dari API. project ini dikerjakan saat saya berada di kelas 11, dengan fokus pada penerapan HTML, CSS, dan JavaScript untuk membangun galeri film yang dinamis.

Web ini memanfaatkan API dari The Movie Database (TMDb) untuk menampilkan daftar film terbaru dan terpopuler. Pengguna dapat melakukan pencarian film dengan menggunakan fitur pencarian yang responsif. Setiap film ditampilkan dengan poster, judul, dan ratingnya, serta disertai dengan ringkasan singkat yang muncul saat pengguna mengarahkan kursor ke gambar film.

Dengan menggunakan CSS, saya menambahkan gaya yang menarik untuk menciptakan pengalaman visual yang menyenangkan, termasuk penggunaan warna yang kontras dan layout yang responsif agar tampilan tetap optimal di berbagai perangkat. JavaScript digunakan untuk menangani pengambilan data dari API dan interaksi pengguna, seperti menampilkan hasil pencarian dengan cepat dan efisien.

project ini tidak hanya memperkuat pemahaman saya tentang pengembangan web tetapi juga memberikan kesempatan untuk belajar cara bekerja dengan API dan mengelola data dinamis di halaman web.

Simple form internship student

Project ini adalah tugas pembuatan form pendataan siswa magang yang dibuat saat kelas 11. Dalam Project ini, siswa diminta untuk membuat form menggunakan HTML dan CSS untuk mengumpulkan informasi penting mengenai siswa yang akan menjalani program magang. Form ini dirancang dengan beberapa input field seperti nama, absen, kelas, jurusan, dan informasi kontak seperti email dan nomor telepon.

Selain itu, terdapat bagian yang memungkinkan siswa untuk mengunggah CV/Portofolio dan memberikan tanggapan mengenai tempat magang yang diinginkan.

Project ini bertujuan untuk membantu siswa menguasai dasar-dasar pembuatan form, validasi input, dan pengalaman dalam membangun antarmuka pengguna yang sederhana dan fungsional.
